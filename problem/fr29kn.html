  • the length of both arrays is equal to mmm ;
  • each element of each array is an integer between 111 and nnn (inclusive);
  • ai≤bia_i \le b_ia​i​​≤b​i​​ for any index iii from 111 to mmm ;
  • array aaa is sorted in non-descending order;
