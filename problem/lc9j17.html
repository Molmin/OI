#lc9j17. Partial Sorting

题目描述

给定整数 nnn 和质数 MMM。
对于一个 1∼3n1\sim 3n1∼3n 的排列 ppp，你可以进行下列操作：

  • 将 ppp 的前 2n2n2n 个元素从小到大排序。
  • 将 ppp 的后 2n2n2n 个元素从小到大排序。

可以证明任意 1∼3n1\sim 3n1∼3n 的排列都能通过若干次上述操作从小到大排序。
记 f(p)f(p)f(p) 表示将 ppp 从小到大排序所需的最小总操作次数。
你需要求出对于所有 1∼3n1\sim 3n1∼3n 的排列 ppp，其 f(p)f(p)f(p) 总和对 MMM 取模后的值。

输入格式

输入一行两个整数 n,Mn,Mn,M（1≤n≤1061\leq n\leq 10^61≤n≤10​6​​，108≤M≤10910^8\leq M\leq 10^910​8​​≤M≤10​9​​），保证 MMM 是质数。

输出格式

输出一行一个整数表示 f(p)f(p)f(p) 的总和对 MMM 取模后的值。

Consider a permutation †p^\dagger p​†​​p of length 3n3n3n . Each time you can do one of the following operations:

  • Sort the first 2n2n2n elements in increasing order.
  • Sort the last 2n2n2n elements in increasing order.

We can show that every permutation can be made sorted in increasing order using only these operations. Let's call f(p)f(p)f(p) the minimum number of these operations needed to make the permutation ppp sorted in increasing order.

Given nnn , find the sum of f(p)f(p)f(p) over all (3n)!(3n)!(3n)! permutations ppp of size 3n3n3n .

Since the answer could be very large, output it modulo a prime MMM.

†^\dagger​†​​ A permutation of length nnn is an array consisting of nnn distinct integers from $ 1 $ to $ n $ in arbitrary order. For example, [2,3,1,5,4][2,3,1,5,4][2,3,1,5,4] is a permutation, but [1,2,2][1,2,2][1,2,2] is not a permutation (222 appears twice in the array), and [1,3,4][1,3,4][1,3,4] is also not a permutation (n=3n=3n=3 but there is 444 in the array).
