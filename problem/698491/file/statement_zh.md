## 题目描述

在 $n \times n$ 的国际象棋棋盘上放 $n$ 个车，要求满足两个条件：

- 所有的空格子都能被至少一个车攻击到。
- **恰好**有 $k$ 对车可以互相攻击到。

答案对 $998244353$ 取模。

## 输入格式

输入一行两个整数 $n,k$。

## 输出格式

输出一行一个整数，表示答案。

## 数据范围

$1\le n \le 2\times 10^5$，$0 \le k \le  \frac{n(n-1)}{2}$。

## 样例输入输出

```input1
3 2
```

```output1
6
```

```input2
3 3
```

```output2
0
```

```input3
4 0
```

```output3
24
```

```input4
1337 42
```

```output4
807905441
```
