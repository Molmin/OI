#0o7ffs. Color

时间限制：6 s       空间限制：256 MiB       标签： 数学 组合数学 容斥原理 组合计数 ICPC 陕西 2014 

算法难度等级：3       思维难度等级：4       实现难度等级：4


本题来源于：2014 年 ICPC 亚洲（陕西）西安区域赛 Problem F

nnn 朵花，按顺序排成一排。从 mmm 种颜色中选出 kkk 种颜色，给这 nnn 朵花染色，要求相邻的花颜色不同。问共有多少种染色方案？答案对 109+710^9 + 710​9​​+7 取模。

1≤n,m≤1091\leq n,m\leq 10^91≤n,m≤10​9​​，1≤k≤1061\leq k\leq 10^61≤k≤10​6​​，k≤n,mk\leq n,mk≤n,m。

Description

Recently, Mr. Big recieved nnn flowers from his fans. He wants to recolor those flowers with mmm colors. The flowers are put in a line. It is not allowed to color any adjacent flowers with the same color. Flowers iii and i+1i + 1i+1 are said to be adjacent for every iii, 1≤i<n1\leq i < n1≤i<n. Mr. Big also wants the total number of different colors of the nnn flowers being exactly kkk.

Two ways are considered different if and only if there is at least one flower being colored with different colors.

Input

The first line of the input gives the number of test cases, TTT. TTT test cases follow. TTT is about 300300300 and in most cases kkk is relatively small.

For each test case, there will be one line, which contains three integers n,m,kn, m, kn,m,k (1≤n,m≤1091 \leq n, m \leq 10^91≤n,m≤10​9​​, 1≤k≤1061 \leq k \leq 10^61≤k≤10​6​​, k≤n,mk \leq n, mk≤n,m).

Output

For each test case, output one line containing Case #x: y, where xxx is the test case number (starting from 111) and yyy is the number of ways of different coloring methods modulo 109+710^9 + 710​9​​+7.
